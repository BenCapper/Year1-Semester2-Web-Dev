Tippy


Scripts

The following scripts must be linked to the page by placing them below the JQuery scripts. at the bottom of the body tag.

< script src="https://unpkg.com/@popperjs/core@2">
< script src="https://unpkg.com/tippy.js@6">

Stylesheets

To use animations that come with Tippy, the corresponding stylesheet must be included below the scripts added in the last step.

< link rel="stylesheet" href="https://unpkg.com/tippy.js@6/animations/shift-away.css"/>


This imports the shift away animation. To include other animations, just copy this link and change the name of this file to the name of the animation you want to add. (Replace shift-away with scale)



Creation

The next step is to include the data-tippy-content attribute in the element you want a tooltip for, (Replace Tooltip with the text you would like displayed.)



Themes

Custom Themes can be made in CSS.
Target .tippy-box[data-theme~='green'] and enter the style rules. (Replace green with the name of your theme.)



JQuery

The last step is to initialize Tippy and pass in the settings you want to use for customization
There are lots of options to customize the tooltips, including:


tippy('#gallery img', { <======> Inititializes the plugin.
maxWidth: 250, <======> Max width of any tooltip is 250px.
arrow: false, <======> False turns the arrow off, True turns it on.
delay: 350, <======> Tooltip has a 350ms delay before appearing.
animation: 'scale', <======> Applies the animation chosen. (Stylesheet for the chosen option must be included in your page.)
placement: 'bottom', <======> Tooltips will appear below the element.
theme:'blue' <======> Applies the theme you have chosen.
});
